# Corollary and Sard's Theorem (Continued)

> [!INFO] Corollary (restated)
> $Y$ boundaryless, $X, Z \subseteq Y$ submanifolds, $Z$ boundaryless. Then $X \cap Z$ is a submanifold (if $\operatorname{codim}_Y = \operatorname{codim}_Y X + \operatorname{codim}_Y Z$) with boundary $\partial X \cap Z$ if:
> - For all $p \in X \cap Z$: $T_p X + T_p Z = T_p Y$, and
> - If $p \in \partial X$: $T_p \partial X + T_p Z = T_p Y$.
> 
> *(Use $i: X \hookrightarrow Y$.)*

> [!INFO] Sard's Theorem
> Let $f: X \to Y$ be smooth with $Y$ boundaryless. Then almost all $y \in Y$ are regular for $f$ (and for $\partial f$). *(Outside a measure-0 set.)*
